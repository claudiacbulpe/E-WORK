Inicio

¡Hola! ¿Te gustaría aprender acerca de E-Work? En este proyecto interactivo el objetivo es enseñarte los conceptos básicos del mismo. Para empezar, hagamos un pequeño cuestionario para ver cuanto sabes sobre E-Work hasta ahora:

Pregunta

Empecemos por lo  más simple: ¿Qué es el E-Work?  ¡Escoge la respuesta más correcta y la más completa!

Respuestas

Es una herramienta digital que usamos para agilizar nuestro trabajo en la oficina

Es una forma de organización y realización de nuestro trabajo a distancia utilizando las tecnologías de la información y la comunicación (las TICs)

Es el proceso mediante el cual adaptamos el trabajo manual para que sea automatizado y pase a ser totalmente digital. 

Pregunta

¿Cuáles son los tres elementos del E-Work?

Respuestas

El elemento técnico, el elemento digital y el elemento personal

El elemento manual, el elemento digital y el elemento temporal

El elemento técnico, el elemento temporal y el elemento espacial

El elemento lineal, el elemento humano y el elemento tecnológico

Pregunta

En el desarrollo del E-work encontramos seis aspectos. Estos son:

Respuestas

Trabajo, comunicación, ciberseguridad, cambio, espacialidad, distanciamiento

Trabajo personal, comunicación a distancia, seguridad internacional, intercambio de datos, situación actual, distancia entre elementos

Trabajo abierto, comunicación, seguridad, intercambio, espacio, distancia
